Overview

Welcome to the Verti CV Parsing SaaS documentation! Here you'll find everything you need to get started integrating our service into your application.

General Installation

  1. Sign up for an account and obtain your API key.
  2. Review your usage limits and documentation access.
  3. Choose your preferred integration method below.

Angular

npm install verti-cv-parser-angular

Import the module and use the provided components in your forms.

React

npm install verti-cv-parser-react

Use the <CVParser /> component in your React app.
